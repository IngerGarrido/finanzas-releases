# Finanzas Personales (release público)

Distribución de la app de finanzas personales. Para instalar:

1. Subí todo el contenido por FTP/cPanel a la raíz de tu servidor.
2. Configurá el archivo `backend/.env` con los datos de tu base de datos.
3. Las actualizaciones futuras se aplican desde Sistema → Actualizaciones.

Versión interna: `e31641a`
